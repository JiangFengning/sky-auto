 **任务一：和朋友击掌 0/1**

完成方法：

1.点击好友找到牵手上方的击掌 点击与好友完成击掌即可

 **注：一级二级都可以完成**

2.点击[集结向导]剧情节点 找到第一个剧情任务上方的击掌，与[向导]完成击掌即可

 **注：需要完成第一个[表演季]剧情任务**
