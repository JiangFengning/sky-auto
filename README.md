**任务一：给一位朋友送礼 0/1**

完成方法：

1.点击好友找到击掌右边的礼物花费3根普通蜡烛并送给对方爱心后即可完成

2.在 [好友星盘¹] 上随机点击一位已经解锁击掌的好友，找到第三个图标花费3根普通蜡烛完成送礼即可
