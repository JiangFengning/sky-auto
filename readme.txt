 **任务一：和朋友击掌 0/1**

完成方法：

1.点击好友找到牵手上方的击掌 点击与好友完成击掌即可

 **注：一级二级均可完成**

2.在 圆梦村剧场¹ [剧团团长]找到第一个魔法节点上方击掌，与 [剧团团长] 完成击掌即可

 **注：需要完成第一个[表演季]剧情任务**

3.在大树屋找到并点击[集结向导]剧情节点 找到第一个剧情任务上方的击掌，与 [集结向导] 完成击掌即可

 **注：需要完成第一个[集结季]剧情任务**
