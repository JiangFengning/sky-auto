完成方法：

1.点击好友找到牵手上方的击掌 点击与好友完成击掌即可

 **任务三：拯救一只被黑暗藤蔓困住的遥鲲 0/1**

拿出心火蜡烛燃烧黑暗藤蔓拯救遥鲲即可完成

 **2.密林遗迹右侧矿洞前**

 **注：共有两处随机拯救一只即可完成**

找到风行向导 重温剧情任务五 燃烧黑暗植物 拯救遥鲲即可
