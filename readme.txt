 **任务一：给一位朋友送礼 0/1**

完成方法：

1.点击好友 找到击掌右边的礼物 花费3跟蜡烛并送给对方爱心后即可完成

2.在星盘上随机点击一位已经解锁击掌的好友，找到第三个图标 花费3跟蜡烛完成送礼即可

 **注：第一种 二种任选其一即可完成**

 **任务三：收集30点烛光 0/30**

1.领取任务
