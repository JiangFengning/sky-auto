 **注：共有两处随机拯救一只即可完成**

![](https://img.166.net/reunionpub/ds/kol_server/20240802/000550-63ujf0ecob.jpeg)

![](https://img.166.net/reunionpub/ds/kol_server/20240802/000557-asti31qj5p.jpeg)

找到风行向导 重温剧情任务五 燃烧黑暗植物 拯救遥鲲即可
